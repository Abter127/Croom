# Control Room — Case Study

A human oversight interface for enterprise AI agents. This is a React-based portfolio case study exploring risk-proportional oversight for AI governance.

## Quick Start

```bash
npm install
npm run dev
```

The app will open at `http://localhost:3000`.

## Build

```bash
npm run build
```

## Project Structure

- `src/App.jsx` — Main React component with all sections
- `src/main.jsx` — React entry point
- `index.html` — HTML template
- `vite.config.js` — Vite configuration

## Features

- Scroll-linked progress bar
- Intersection observer animations
- Live countdown timers
- Interactive component library (TaxonomyVisualizer, DecisionsSection, etc.)
- Responsive design
- Dark mode with custom color system

## Design System

The interface uses a dark editorial theme:
- **Primary Accent**: Indigo (`#818CF8`)
- **Semantic Colors**: Red, Amber, Green for risk levels
- **Typography**: Outfit (body) + Space Mono (mono)
